Gurney Guide

Welcome to Gurney Guide

Gurney Guide is an excellent tool that can make navigating a hospital much easier.

You can upload a DXF file of your hospital's floorplan, view it, and find the best path to the room of your choice.

Getting Started

To get started, you can upload a DXF file of your hospital's floorplan in the Upload or View Floorplan tab.

Once the file is uploaded, you can see the ID of the floorplan and view or delete it using the buttons adjacent to the floorplan.

Then, you can go to the Pathfinder tab to find the best path to a specific room by entering the floorplan ID along with the room number.

Additional Information

If you encounter issues, ensure your DXF file is properly formatted and follows the required specifications. For further assistance, contact support.
